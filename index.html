Start

Setting up Font WSO2 can be as simple as adding two lines of code to your website!

  1. Download the latest font pack and copy files as below structure. 
    root/ [ where your page is ]

│   ├── fontwso2-extend.css
└── fonts/

    ├── fontwso2.ttf
